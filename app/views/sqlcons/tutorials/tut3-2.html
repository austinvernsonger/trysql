Ch3-2 : Insert (cont.)

Now let's get some more practice with the insert. What if we wanted to add more than one row per insert? We could include multiple rows by adding comma-seperation to our values clause. Let's say we wanted to add two teachers to our database:
insert into teachers VALUES (15, 'Jack', 'Lincoln'), (16, 'Jose', 'Lopez')
Great! Now lets check and see that those rows were actually inserted.
select * from teachers t order by t.fname
