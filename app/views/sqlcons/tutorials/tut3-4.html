Ch3-4 : Update

Our remaining command to learn is the update statement. If you wanted to alter a particular column value for a table, you could simply apply an update rather than deleting, then adding that row with the new value. Let's say we wanted to alter one of our course titles, we could use the following.
update courses set title = 'New style Spanish' where cnum = 126
Now check the updated table.
select * from courses
